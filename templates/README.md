# 시작 템플릿 안내

이 폴더는 웹페이지 제작이 처음인 교사가 바로 열어보고 구조를 이해할 수 있도록 만든 예시 파일 모음입니다.

## 포함 템플릿

| 폴더 | 용도 | 추천 상황 |
|---|---|---|
| `guide-page` | 수업 안내 페이지 | 활동 순서, 자료 링크, 제출 방법을 한 주소에 모을 때 |
| `quiz-activity` | 간단 퀴즈 활동 | 학생이 버튼을 눌러 정답과 해설을 확인하게 할 때 |
| `form-hub` | 응답 연결 허브 | 웹페이지는 안내판으로 쓰고 제출은 Forms나 Tally로 받을 때 |
| `map-activity` | 지도 탐색 활동 | 지점 버튼을 눌러 지역 설명과 탐구 질문을 확인하게 할 때 |

## 사용 전 확인

- 예시 링크의 `#`는 실제 자료 링크나 제출 링크로 바꿔야 합니다.
- 학생 이름, 학번, 성적, 얼굴 사진은 템플릿에 직접 넣지 않는 것을 기본으로 합니다.
- 이미지나 PDF를 추가하면 배포할 때 해당 파일이 들어 있는 폴더도 함께 올려야 합니다.
- 실제 수업에 쓰기 전 학생 기기에서 열어보고, 공개 범위와 제출 링크 권한을 확인하세요.
